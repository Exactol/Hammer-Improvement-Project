# Hammer-Improvement-Project
A project dedicated to improving the Valve Hammer Editor through DLL injection

# Planned features: 

-Plugin system

-Map templates

-Crash logging to track down the sources of Hammer's numerous crashes

-Debugging tools

# Completed features:
